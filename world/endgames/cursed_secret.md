# Endgames — Cursed / Secret

See `../schemas.md` for record format.

The weird-shaped endings. Gated by unusual seed combinations or `craze_unhinged` mode. Some are dark, some are absurd, all of them break the Bloomberg-post-mortem frame on purpose — the long read for these uses the New Yorker fact-meets-fiction register, or pivots into outright absurdism. Share card is the wildcard slot. Some produce a `cult_communique` artifact instead of standard rehabilitation copy.
